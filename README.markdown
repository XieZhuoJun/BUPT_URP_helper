# BUPT_URP_helper
北邮选课助手

Features：
--------------

* OCR识别验证码自动登陆
* 输入课程序号进行课程查找和选课
* 挂机抢课
* 自定义选课尝试间隔
* 自定义选课尝试次数

Pre-requirements
--------------
* Python 3+
* python依赖：pytesseract, PIL
* 非python依赖：tesseract https://github.com/tesseract-ocr/tesseract

Pre-set
---------------

* 在URP-helper.py文件中输入你的学号和教务系统密码
